### âš¡ COMPONENT 2: PRIORITY ENGINE (priority_engine.py)

**What It Does:**
Converts AI analysis into numerical priority score (0-100)

**Scoring Formula:**
```
TOTAL SCORE = Weighted Sum + Bonuses

Base Urgency (30%):
  - AI urgency score (0-10) Ã— 10 Ã— 0.30
  - Example: 10/10 Ã— 10 Ã— 0.30 = 30 points

Time Sensitivity (25%):
  - Medical/rescue: 50-100 points
  - Shelter at night (6PM-6AM): 100 points (CRITICAL)
  - Food/water: 30 points
  - Example: Medical Ã— 0.25 = 12.5 points

Vulnerable Groups (20%):
  - Baby/infant/newborn: 70 points (highest!)
  - Children: 40 points
  - Pregnant: 40 points
  - Elderly: 35 points
  - Example: Baby Ã— 0.20 = 14 points

Immediate Danger (15%):
  - Life-threatening detected: 100 points
  - No immediate danger: 0 points
  - Example: TRUE Ã— 0.15 = 15 points

People Count (10%):
  - 1 person: 40 points
  - 2-3 people: 55-70 points
  - 5+ people: 100 points
  - Example: 2 people Ã— 0.10 = 4 points

BONUSES:
+ Extended Duration Bonus (food/water for "2 days"): +12 points
+ Baby + Immediate Danger: +15 points (CRITICAL combo)

FINAL SCORE MAPPING:
  80-100 = CRITICAL (red)
  60-79  = HIGH (orange)
  40-59  = MEDIUM (yellow)
  0-39   = LOW (blue)
```

```

**Key Innovations:**
1. **Time-Aware**: Nighttime shelter requests get priority boost
2. **Baby Focus**: Babies prioritized over general children

def find_nearby_donors(affected_location, resource_type, max_distance=15):
    matches = []
    
    # Get all available donations of this resource type
    for donation in all_donations:
        if donation.resource_type == resource_type:
            # Calculate distance
            distance = haversine(affected_location, donation.location)
            
            # If within radius
            if distance <= max_distance:
                # Estimate travel time (20 km/h in disaster)
                travel_time = (distance / 20) * 60  # minutes
                
                matches.append({
                    "donor": donation,
                    "distance_km": distance,
                    "travel_time_mins": travel_time
                })
    
    # Sort by closest first
    return sorted(matches, key=lambda x: x['distance_km'])
```

- Primary Mode: Gemini AI extracts context, understands variations
- Fallback Mode: Regex pattern matching for keywords
- Result: System ALWAYS works, with or without AI
