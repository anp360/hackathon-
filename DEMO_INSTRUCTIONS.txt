## ğŸ› TROUBLESHOOTING

### Issue 1: Server Won't Start

**Error:** `Address already in use` or `Port 5000 is already allocated`

**Solution:**
```powershell
# Windows: Kill process on port 5000
netstat -ano | findstr :5000
taskkill /PID [PID_NUMBER] /F

# Or run on different port
python app.py --port 5001
# Then open http://localhost:5001
```

---

### Issue 2: Dashboard Blank/Not Loading

**Symptoms:** White screen, no content

**Solutions:**
1. **Check Browser Console:**
   - Press F12 â†’ Console tab
   - Look for JavaScript errors
   - Share errors if asking for help

2. **Hard Refresh:**
   - Windows: Ctrl + Shift + R
   - Mac: Cmd + Shift + R

3. **Try Different Browser:**
   - Chrome (recommended)
   - Firefox
   - Edge

4. **Check Server Logs:**
   - Look at terminal where app.py is running
   - Any Python errors shown?

---

### Issue 3: Messages Not Appearing

**Symptoms:** Submit message, but doesn't show up

**Solutions:**
1. **Check Database File:**
   ```powershell
   # Verify messages are being saved
   type data\messages_db.json
   ```

2. **Click Refresh:**
   - Click ğŸ”„ Refresh button
   - Or reload page (F5)

3. **Check API Response:**
   - Open browser DevTools (F12)
   - Network tab
   - Submit message
   - Look for `/api/submit_message` request
   - Check response

---

### Issue 4: Media Upload Fails

**Symptoms:** "Error uploading media" or "File too large"

**Solutions:**
1. **Check File Size:**
   - Images: Max 5MB
   - Videos: Max 50MB (~30 seconds)
   - Compress if too large

2. **Check File Format:**
   - Supported: JPG, PNG, MP4, MOV
   - Unsupported: RAW, TIFF, AVI, MKV

3. **Check Permissions:**
   ```powershell
   # Ensure data/media folder exists and writable
   mkdir data\media
   mkdir data\media\thumbnails
   ```

---

### Issue 5: Gemini API Not Working

**Symptoms:** "AI processor not initialized" error

**Solutions:**
1. **Using Fallback Mode:**
   - System works WITHOUT API key!
   - Uses keyword-based analysis
   - Still provides accurate priority scoring

2. **If You Want Real AI:**
   ```powershell
   # Set environment variable
   $env:GEMINI_API_KEY="your-actual-api-key"
   
   # Restart server
   python app.py
   ```

3. **Verify API Key:**
   - Check format: Starts with "AIza..."
   - No spaces or quotes
   - Valid and active key
